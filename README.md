# Construct FM — Estimate Generator

Internal tool for building accurate construction estimates from the Construct FM verified rate card.

## Features

- **Full Rate Card**: 50+ rates across 11 categories (partitions, decoration, flooring, roofing, drainage, etc.)
- **Line Item Builder**: Click rates to add, edit quantities/rates inline, add custom items
- **Auto-Calculations**: Prelims %, VAT, grand total — all live
- **Print/PDF**: Professional branded estimate output with logo, terms, and footer
- **Construct FM Branding**: Dark theme with orange accent (#f2a735)

## Rate Sources

All rates extracted from real Construct FM quotations:
- CQ-010156: SMR Corner Extension
- CQ-010167: Rhino Enclosure (Longleat)
- CQ-010168: Gully Drainage (Longleat)
- TEC1096MH: Landau Marine full office refurb (£89,850)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
